# OrganisationforJunior
junior quackathon

To run the program:
Using an ide or powershell move into the OrganisationforJunior folder
run 'node server.js'
go to http://localhost:3000

to create an account input username and password then click Sign in (register)
then input your information again and click Log in.
Duplicate usernames can't be used.

On home page to confirm the times you need to press the submit button for every day
then the click 'Find Matches'.

To add friends click add friend on the matches page,
to remove press the same button. it is also possible to remove friends on the Friends page.

Friends page display the list of friends.
